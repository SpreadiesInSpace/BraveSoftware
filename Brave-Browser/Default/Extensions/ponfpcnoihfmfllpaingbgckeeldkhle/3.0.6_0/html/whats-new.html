What's new in Enhancer for YouTube™ v3.0.0

Yes, you read it right, version 3 is finally here!
You are now using the best version ever created

Bugs

All reproducible issues have been fixed.

If you find a new bug it means YouTube has changed something, or you have been selected to test something new.

In any case, it's now possible to copy debug info that will help me reproduce and fix issues (more info here).

If you're curious, do it at least once just to see how many experiments are tested when you visit YouTube (check the “experiment_flags” in the debug info), you will realize how difficult it is to maintain an extension like this one.

Improvements

(non-exhaustive list)
  • Code fully rewritten: increased overall performance and smoother user experience
  • Redesigned options page
  • New theme: created my own theme and variants because the 3rd party theme “YouTube-DeepDark” is not compatible with YouTube's “Ambient mode” and other recent design changes
  • Preferred quality fallback: “auto” if your preferred quality is “480p”, “360p”, “240p” or “144p”, otherwise the highest available quality
  • Mini player: now supports common video aspect ratios (4:3, 16:9, 18:9, 21:9, 32:9) and can also be positioned in the top center and bottom center
  • Expanded player: now supports the fixed chat panel
  • Video filters panel: accessibility has been improved, you can now use arrow keys when a filter has focus
  • Compatible with the upcoming—but still experimental—new layout (see the “Experiments” section below)
  • Less prone to fatal errors when YouTube will make important changes

Experiments

YouTube is continuously experimenting UI changes and other things that can prevent Enhancer for YouTube™ from working as expected, so by default the extension blocks some experiments, but that can cause unexpected issues on YouTube too, this is why with version 3 it's now possible to allow experiments.

But unlike other features, this one cannot be implemented as a setting in the options page because it must be executed immediately when the page load begins, and at this stage your preferences cannot be retrieved... More info here if you want to give it a try, it's really easy.

Important: Reload all the YouTube tabs that were open before receiving the update or you will run into issues...
